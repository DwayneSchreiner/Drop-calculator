# 🎯 Fortnite Drop Calculator

Interactive drop calculator voor **Fortnite Chapter 7 Season 2 — Showdown (Golden Coast)**.

Gebouwd als één HTML-bestand, geen installatie nodig.

## 🗺️ Live

👉 [Open Calculator](https://jouwusername.github.io/fortnite-drop-calc)

## ✨ Features

- Interactieve Golden Coast map met alle Ch7S2 POIs
- Klik-om-te-droppen: stel busroute in en kies je landingslocatie
- Berekent sprong-timing, vrije val, glijfase en max bereik
- Animated bus langs de route
- Tactisch advies per drop type (hot / mid / cold)
- Werkt op telefoon en desktop

## 🕹️ Gebruik

1. **Tik voor Bus Start** — zet het beginpunt van de bus
2. **Tik voor Bus Einde** — teken de busroute
3. **Tik voor Drop** — kies je landingslocatie
4. Resultaten verschijnen direct onderaan

## ⚙️ Instellingen
